### 5. LongStraw: Long-Context RL Beyond 2M Tokens under a Fixed GPU Budget

> 评分 9/10 · 长上下文 RL 后训练突破 2M tokens，upvotes 极高，直接解决 agent 长轨迹训练痛点，方法可复现

## 本周趋势

本周行业焦点高度集中于智能体（agent）的工程化落地，安全围栏与上下文工程成为两大核心支柱。Anthropic 公开了跨产品 agent 安全围栏的实战方法，直面“爆炸半径”这一生产级痛点，标志着 agent 安全从概念走向可复现的工程实践。同时，上下文工程被重新定义为一种需要精心构建的“工程产物”，强调通过结构化信息与动态管理提升 agent 在复杂多步任务中的稳定性，这反映出行业正从依赖模型能力转向系统化地设计 agent 运行环境。此外，长上下文强化学习训练取得突破，LongStraw 通过“分离评估与回放”策略，在固定 GPU 预算下实现了百万 token 级别的 RL 后训练，为生产级长轨迹 agent 的策略优化提供了可行路径，预示着 agent 训练将不再受限于短序列泛化的妥协方案。

1. 安全围栏设计：可以谈如何在实际应用中通过权限分级、工具调用审计和运行时监控来限制 agent 的破坏半径，这直接关系到架构选型与风险控制策略，体现对生产环境安全责任的工程理解。
